Help Section - Bruce Virtual Assistant

Welcome to the Help Section for Bruce, your virtual assistant designed to automate tasks and simplify your life. This guide provides a detailed explanation of all features, commands, and examples to help you use Bruce effectively.

Modes of Operation

Bruce operates in two modes:

  • Basic Mode: Handles simple tasks such as opening files, playing music, and checking the time.
  • Advanced Mode: Handles complex queries, integrates AI (ChatGPT), and provides advanced task management.

Features and Commands

1. Time and Date

Bruce can provide the current time and date.

Command: "What is the time?"

Command: "Show me today’s date."

Response: "The current time is 10:30 AM" or "Today's date is March 3, 2025."

2. File Management

Bruce can manage your files with commands to open, delete, move, and rename files.

  • Command: "Open file example.txt."
  • Command: "Delete file example.txt."
  • Command: "Move file example.txt to Documents."

3. Media Control

Bruce can control your media playback, such as playing or pausing music and adjusting the volume.

  • Command: "Play my favorite playlist."
  • Command: "Pause the music."
  • Command: "Increase the volume."

4. Web Search

Bruce can search the web for information.

  • Command: "Search for the capital of France."
  • Command: "Find the latest news about technology."

5. Reminders

Set reminders for tasks or events.

  • Command: "Set a reminder for my meeting at 3 PM."
  • Command: "Remind me to water the plants tomorrow."

6. Weather Updates

Bruce provides the current weather and forecasts.

  • Command: "What’s the weather like today?"
  • Command: "Will it rain tomorrow?"

7. Quick Dictionary

Look up definitions or synonyms.

  • Command: "Define the word algorithm."
  • Command: "What are the synonyms of happy?"

8. Notepad Integration

Create, edit, and save notes.

  • Command: "Open notepad."
  • Command: "Save this note as tasks.txt."

9. Task List Management

Manage your task list with ease.

  • Command: "Add buy groceries to my task list."
  • Command: "Show my tasks."
  • Command: "Mark buy groceries as done."

10. Advanced NLP Queries

Handle complex queries using AI (ChatGPT).

  • Command: "Summarize this article for me."
  • Command: "Explain quantum mechanics in simple terms."

11. Context-Aware Suggestions

Bruce can provide recommendations based on your activity and preferences.

  • Command: "Suggest a movie for tonight."
  • Command: "Play some relaxing music."

12. System Commands

Control your computer using voice commands.

  • Command: "Shut down the computer."
  • Command: "Take a screenshot."
  • Command: "Display system information."

13. Multi-language Support

Use Bruce in different languages for speech-to-text and text-to-speech.

  • Command: "Switch to French."

14. Application Shortcuts

Open frequently used applications with shortcuts.

  • Command: "Open Chrome."
  • Command: "Launch Photoshop."
